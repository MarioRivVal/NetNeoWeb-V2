Student Management

Este proyecto fue desarrollado para un educador que me confió la tarea. Siguiendo la arquitectura Modelo-Vista-Controlador (MVC) y aprovechando el poder de las clases, diseñé esta aplicación. Utiliza de manera fluida IndexedDB, convirtiéndola en una Progressive Web App (PWA) totalmente funcional.

El objetivo principal es facilitar la gestión de los registros de estudiantes, incluida información detallada y seguimiento de pagos. Periódicamente, cada tres meses, identifica y genera automáticamente facturas para los estudiantes que han realizado pagos, simplificando el proceso administrativo para el docente.

That's Amore FoodOrder

Pruebalo

La aplicación para tomar pedidos está diseñada pensando en la eficiencia y conveniencia. Utilizando una robusta arquitectura MVC y aprovechando el poder de las clases, esta aplicación gestiona de manera fluida pedidos y mesas, garantizando un funcionamiento sin problemas para el personal. Con la integración de archivos JSON , Promises y Local Storage, ordenar comida y bebidas se convierte en algo sencillo, incluso estando en movimiento con la accesibilidad móvil.

Además, cada pedido se registra automáticamente con la fecha y hora actual, proporcionando actualizaciones en tiempo real para una mayor precisión. Además, hemos añadido un toque de personalización al permitir que los usuarios ajusten el porcentaje de propina según su preferencia.
